## Веб-додаток з Socket-сервером та MongoDB на Docker

Коротка інструкція для запуску та перевірки проекту

### Запуск

1. Команди для збірки та запуску контейнерів:

- `docker-compose up --build`
- `docker-compose up -d --build` (для фонового режиму)

2. Для зупинки проекту:

- `docker-compose down`

### Як користуватись

1. **Веб-сайт**: Відкрийте http://localhost:3000 у вашому браузері
2. **Відправка повідомлення**:
- Перейдіть на http://localhost:3000/message.html
- Заповніть форму та натисніть "Send"
- Дані будуть збережені в MongoDB
3. **Сторінка 404**: Перейдіть на http://localhost:3000/something, щоб побачити сторінку помилки
